Fortran kodlarını derlemek
==========================
.. code-block:: bash
   
   $ gfortran -c freefall.f90 		// Output object file : freefall.o
   $ gfortran -o freefall freefall.f90 	// Output executable file : freefall
   $ gfortran freefall.f90 		// Output executable file : a.out
